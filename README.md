# CV Creator App

The goal of this Odin project was to build a simple and user-friendly CV creator application using the React library. The app allows users to quickly generate a structured resume by filling out a form and instantly previewing the result.
To set up the project environment, the Vite tool was used, providing fast performance, HMR and a modern development experience.
In order to enable users to download their created CV as a PDF file, the html2pdf.js package was integrated.

### Features

- Clean and minimalistic user interface for better focus and usability
- Live preview of the generated resume
- Export CV as a PDF file
