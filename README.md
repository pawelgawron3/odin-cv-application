# 🧾 CV Creator App

The goal of this Odin project was to build a simple and user-friendly CV creator application using the React library. The app allows users to quickly generate a structured resume by filling out a form and instantly previewing the result in real time.

To set up the project environment, the Vite tool was used, providing fast performance, HMR and a modern development experience.

In order to enable users to download their created CV as a PDF file, the html2pdf.js package was integrated.

---

## ✨ Features

- Clean and minimalistic user interface focused on usability
- Live preview of the CV while typing
- A4-style layout preview on the right side of the screen
- Fast development setup powered by Vite
- Export CV as a PDF file using `html2pdf.js`

---

## 🛠️ Tech Stack

- HTML
- CSS
- Vite
- React with React Hooks (`useState`)
- html2pdf.js

---

## 📂 Project Structure

The project is structured in a clean and maintainable way, with separated responsibilities.

---

## 📄 PDF Export

The CV can be exported as a PDF file using html2pdf.js. The layout is designed to mimic a real A4 document, ensuring the exported file looks clean and professional.
